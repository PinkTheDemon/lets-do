`quick_validate.py` 是 skill 的快速结构校验脚本，用来在安装前检查这个 skill 是否“长得合规”。

它主要会检查这些内容：

- `SKILL.md` 的 YAML frontmatter 是否存在、格式是否正确
- frontmatter 里是否至少有 `name` 和 `description`
- skill 名称是否符合规则，比如只能是小写字母、数字、连字符，长度不能超限
- 目录结构是否基本合法

它不是运行 skill 逻辑本身，也不是功能测试，更像“静态体检”。
